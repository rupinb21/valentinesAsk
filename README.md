# Valentine's Ask

A simple single page webpage to send your valentine.

## Features

- Beautiful responsive design that works on both desktop and mobile
- Interactive Yes/No buttons with playful responses
- Romantic theme with love heart favicon
- Smooth animations and transitions

## Usage

Simply open the webpage and share it with your valentine! They'll be greeted with a special question and delightful responses regardless of their answer.

## Technologies

- React with Vite for fast development
- Responsive CSS design
- Modern web development practices
